This is a collected repo. Follow the installation guide:
<!-- -------------------------------------------------------------------------------- -->

create env with python==3.9.7 and clone the repo and install the followings.

use command:
conda create --name labelimg python==3.9.7 -y
git clone https://github.com/arunavade99/labelimg.git
conda activate labelimg

conda install pyqt=5
conda install -c anaconda lxml
